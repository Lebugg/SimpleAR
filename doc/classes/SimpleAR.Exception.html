\SimpleAR

\SimpleARException

This class extends Exception class.

The main goal is to provides a default status code. Also, it allows us to distinguish our exceptions of classic system's exceptions.

It is also able to handle PDO exceptions.

sqlException()

sqlException(\SimpleAR\PDOException $oEx,  $sQuery, string $sMessage) : \SimpleAR\ApiException

Instantiates an ApiException from an PDOException.

You can use it this way:

 throw ApiException::sqlException($oEx, 'Database error!');
