parseConditionArray()

parseConditionArray( $aArray)

We accept two forms of conditions: 1) Basic conditions: <code> array( 'my/attribute' => 'myValue', .

..

 )

2) Conditions with operator:

array( array('my/attribute', 'myOperator', 'myValue'), ... )

Of course, you can combine both form in a same condition array.

By default, conditions are linked with a AND operator but you can use an OR by specifying it in condition array:

array( 'attr1' => 'val1', 'OR', 'attr2' => 'val2', 'attr3' => 'val3, )

This correspond to the following exhaustive array:

array( 'attr1' => 'val1', 'OR', 'attr2' => 'val2', 'AND', 'attr3' => 'val3, )

You can nest condition arrays. Example:

array( array( 'attr1' => 'val1', 'attr2' => 'val2', ) 'OR', array( 'attr3' => 'val3, 'attr1' => 'val4, ) )

So we come with this condition array syntax tree:

condition_array: array( [condition | condition_array | (OR | AND)] * ); condition: [ 'attribute' => 'value' | array('attribute', 'operator', 'value') ] attribute: operator: value:

Operator: =, !=, IN, NOT IN, >, <, <=, >=.
