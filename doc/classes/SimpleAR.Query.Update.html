\SimpleAR\Query

\SimpleAR\QueryUpdate

This class handles UPDATE statements.

It defines several static methods to build queries:

  • Query::insert();
  • Query::select();
  • Query::count();
  • Query::update();
  • Query::delete();

$_bIsCriticalQuery : bool

Is this query class critical?

  • DELETE queries {@see SimpleAR\Query\Delete}
